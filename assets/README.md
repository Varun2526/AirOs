# `assets/`

## Why This Exists

Central location for all static media used by the application, documentation, or marketing.

## What Belongs Here

- Application icons and logos
- UI images and sprites
- Sound effects and audio files
- Demo GIFs and screenshots for documentation
- Design mockups and wireframes

## What Does NOT Belong Here

- **Dataset images** → `data/`
- **Model weight files** → `models/`
- **Documentation text** → `docs/`
- **Recorded gesture sessions** → `recordings/`
- **Generated or temporary files** → should be git-ignored
